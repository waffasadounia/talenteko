# Fixtures TalentÉkô (Base de test)

Ces fixtures ont été développées pour la génération automatisée de données
pendant la phase de développement.

Elles couvrent :

- Catégories
- Utilisateurs de test
- Profils
- Annonces (listings)
- Images d’annonce
- Messages
- Fils de discussion (threads)
- Échanges
- Avis (reviews)
- Favoris

Elles sont placées ici dans le dossier `docs/` spécialement pour le jury.
Symfony ne les charge pas automatiquement.
